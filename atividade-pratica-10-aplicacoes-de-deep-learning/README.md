Este repositório reúne a execução da **Atividade Prática 10**, desenvolvida para demonstrar, na prática, uma aplicação de **Deep Learning** por meio do **Teachable Machine**, ferramenta online do Google que permite treinar modelos de aprendizado de máquina de forma simples, visual e sem necessidade de programação.

## Objetivo do repositório

O objetivo principal deste repositório é documentar o desenvolvimento de um modelo capaz de classificar imagens em duas categorias: **gato** e **cachorro**. A proposta permite observar, na prática, como um modelo treinado com exemplos visuais consegue reconhecer padrões e identificar corretamente uma nova imagem.

Além da atividade principal, o repositório também inclui um relatório complementar sobre aplicações de Deep Learning, abordando temas como:

* reconhecimento de imagens;
* processamento de linguagem natural;
* análise de sentimentos;
* detecção de fraudes;
* análise de preços;
* sistemas de recomendação;
* modelos generativos;
* modelos sequenciais;
* detecção e segmentação de imagens.

## Contexto da atividade

A atividade foi proposta para verificar o uso de Deep Learning na prática, com base em estudos e na aplicação de software específico. Para isso, foi utilizado o **Teachable Machine**, que permite criar classificadores a partir de exemplos fornecidos pelo usuário.

O experimento consistiu em inserir imagens de **gatos** e **cachorros**, treinar o modelo e testar sua capacidade de reconhecimento em uma imagem inédita.

## Ferramentas utilizadas

* **Teachable Machine** – treinamento do modelo de classificação;
* **GitHub Pages** – publicação da versão web do projeto;
* **Python** – uso complementar para testes e validação do modelo exportado;
* **keras / tf_keras** – carregamento do modelo treinado.

## Resultados esperados

Com esta atividade, espera-se demonstrar que redes neurais podem aprender padrões visuais mesmo com um conjunto pequeno de dados. O modelo treinado deve ser capaz de:

* distinguir imagens de gatos e cachorros;
* reconhecer uma nova imagem com boa confiança;
* evidenciar a aplicação prática de técnicas de Deep Learning em visão computacional.

## Organização do repositório

Este repositório pode conter, entre outros itens:

* arquivos do modelo treinado exportado pelo Teachable Machine;
* página web para testes em tempo real;
* relatório descritivo da atividade;
* materiais de apoio relacionados ao conteúdo estudado.

## Conclusão

Este projeto mostra, de forma objetiva, como o Deep Learning pode ser aplicado em uma tarefa simples de classificação de imagens. Mesmo em um cenário básico, a atividade evidencia a capacidade do modelo de identificar padrões e realizar previsões úteis a partir de dados visuais.
